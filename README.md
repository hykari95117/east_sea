# 시놉시스(Synopsis)
- 2000년대 초반
- 직장 동료 K에게 자신의 고향인 동해를 여행 장소로 추천해준 C
- 하지만 동해로 여행을 간 K의 행적이 묘연해졌댜.
- 경찰은 K의 행방을 추적하지만 그는 현금만 쓰고 다녔고 핸드폰도 들고가지 않았기에 수색에 어려움을 겪는다.
- 이제 K의 행방을 추적할 수 있는 유일한 단서는 C가 K에게 추천해준 여행 경로와 장소 뿐
- 어쩔 수 없이 경찰은 C에게 협조를 요청하고 C 또한 마지 못해 이를 동행한다.
- 과연 K의 행방은 어떻게 된 것이며, C는 어떠한 이유로 그 장소와 경로를 추천해준 것일까
